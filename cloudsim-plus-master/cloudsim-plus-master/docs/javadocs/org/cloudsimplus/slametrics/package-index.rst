org.cloudsimplus.slametrics
===========================

Provides classes to load SLA contracts from JSON files, according to the format defined by the \ `AWS Cloudwatch <http://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/viewing_metrics_with_cloudwatch.html>`_\ .

A JSON file can be read using a method such as the \ :java:ref:`org.cloudsimplus.slametrics.SlaContract.getInstance(java.lang.String)`\ .
